# jquery.resolution
Simple Helper for Web Designer

Demo : <a href="http://jsfiddle.net/adyoi/3jnfh4ue/" target="_blank">http://jsfiddle.net/adyoi/3jnfh4ue/</a>

# How to use ?

	$('#yourdiv').resolution({
	
		opacity		: 0.5,
		color		: "white",
		background	: "black",
		width		: 300
		
	});
