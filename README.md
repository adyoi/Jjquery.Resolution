# Jquery Display Resolution
Simple Helper for Web Designer

Demo : <a href="https://jsfiddle.net/adyoi/3jnfh4ue/" target="_blank">https://jsfiddle.net/adyoi/3jnfh4ue/</a>

# How to use ?

	$('#resolution').displayresolution({
    		sep		: " | ",
    		font		: "Arial",
    		size		: "12px",
		color		: "#fff",
		background	: "#000",
		width		: 300,
		opacity		: 0.5,
	});
